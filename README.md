# IdeaBlog

本科的毕业设计项目。基于SSM框架实现。做得比较简陋。

## 权限划分
- **游客**访问可浏览博客，提交反馈意见，注册为普通用户；

- **普通用户**除具备游客权限外，还可发表或收藏博客，发表或回复评论，关注用户，上传并分享文件，管理自己的博客和文件，修改个人资料；

- **管理员**除具备普通用户权限外，还可对整个博客系统进行管理和维护，包括系统日志、博客、文件、反馈、用户等。
